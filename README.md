# project-calculator

Input numbers and operations using the buttons, calculate and display the result on the screen. Theres four operations: addition, subtraction, multiplication, and division.

## Technologies Used

- HTML
- CSS
- JavaScript

## How To Use

1. Click the number buttons to type a number
2. Click an operator button (+, -, \*, /) to choose what to do
3. Type the second number
4. Click = to see the result
5. Click **clear** to start over
6. Click **←** to delete the last digit
